## üöÄ Getting Started

Follow these steps to run the project locally:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/portfolio-website.git](https://github.com/YOUR_USERNAME/portfolio-website.git)
    ```

2.  **Navigate to the folder:**

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Start the development server:**
    ```bash
    npm run dev
    ```
